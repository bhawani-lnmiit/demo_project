# Simple Interest Calculator

This project contains a simple Bash script that calculates Simple Interest based on user input.

## Purpose
The purpose of this project is to demonstrate the use of Git, GitHub, and Bash scripting by creating a simple interest calculator.

## How It Works
The calculator takes the following inputs from the user:
- Principal amount
- Rate of interest
- Time period (in years)

It then calculates the Simple Interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100
